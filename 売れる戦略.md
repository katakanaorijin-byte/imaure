# 💰 売れる戦略|「底値レーダー」でお金を稼ぐ方法

正直なところから始めます。**自動更新サイトを置いただけでは、ほぼ稼げません。**
理由は単純で、誰もサイトの存在を知らないからです。そして自動生成だけのサイトはGoogle検索で上位に出にくい、という現実もあります。

ただし、このサイトには大きな武器があります。
**「維持コストがゼロ(お金も時間も)」** ということです。
普通のアフィリエイトは記事を書き続けないと止まりますが、このサイトは放っておいても毎日新鮮です。
だからあなたのやることは1つだけ。**「人をサイトに連れてくること」に全時間を使えます。**

---

## 収益の仕組み(おさらい)

```
人がサイトに来る → 商品リンクをタップ → 30日以内に楽天で何か買う → 代金の2〜4%があなたに
```

ポイントは「**リンクを踏んだ人が買った"別の商品"でも報酬になる**」こと。
つまり商品を売り込む必要はなく、**「楽天を開かせる」だけで成果のチャンス**があります。
「いま売れてるものランキング」はまさにこの"つい開きたくなる"入口として優秀です。

目安:1クリックあたりの期待値はだいたい1〜5円程度。**月1万円を目指すなら月3,000〜1万クリック**が必要、というイメージで考えてください。

---

## 戦略①(最重要):X(旧Twitter)で毎朝発信する

自動更新サイトと一番相性がいいのがSNSです。検索エンジンと違って、新規アカウントでも今日から人を呼べます。

**やり方**
1. サイト専用のXアカウントを作る(名前は「予約開始レーダー|ホビー新着予約速報」など)
2. 毎朝、サイトの更新を見て**1位〜3位だけを紹介するポストを1本**投稿
   - 例:「6/11の楽天売れ筋🥇 ◯◯がまた1位。レビュー2万件超え。全ランキングはこちら→(サイトURL)」
3. これを**毎日続ける**。1日3分の作業です

**伸ばすコツ**
- 「なぜ売れてるのか」を一言添える(セール中、テレビで紹介、季節要因 など)
- 楽天スーパーSALE・お買い物マラソン期間(毎月ある)は投稿頻度を上げる。**この期間が最大の稼ぎどき**
- 固定ポストにサイトURLを置く

> Xの規約上、アフィリエイトリンクを直接大量に貼るのは嫌われやすいので、**「サイトに誘導 → サイト内のリンクから楽天へ」**の流れが安全で効果的です。

## 戦略②:検索(SEO)は「ニッチ特化」で拾う

「楽天 ランキング」のような大きいキーワードでは絶対に勝てません。勝ち筋は**ジャンルを絞ること**です。

**おすすめの改造(generate.pyのGENRESを書き換えるだけ)**
- 「キャンプ用品の売れ筋だけ」「韓国コスメだけ」「ふるさと納税だけ」「猫用品だけ」のような**1テーマ特化サイト**にする
- サイト名も合わせて変える(例:「キャンプいまウレ」)
- 特化すると「◯◯(ジャンル名) 売れ筋」「◯◯ ランキング 今」のような検索で拾われる可能性が出てきます

**さらに効くひと手間**
- 月に1〜2回でいいので、サイトに**自分の言葉の文章を足す**(「今月の傾向まとめ」「1位の商品を実際に買ってみた」など)。検索エンジンは"人間が書いた独自情報"を評価します。GitHub上でindex.htmlに直接書くより、`generate.py` のフッター文を充実させるのが簡単です

## 戦略③:時期で稼ぐ(カレンダー運用)

楽天は「イベントで売れる」モールです。以下の時期はクリック単価も購買率も跳ね上がります。

| 時期 | イベント | やること |
|---|---|---|
| 3・6・9・12月 | 楽天スーパーSALE | SNS投稿を1日2〜3回に増やす |
| 毎月1回 | お買い物マラソン | 同上 |
| 5の0のつく日 | ポイント5倍 | 「今日は0のつく日」と添えて投稿 |
| 11〜12月 | ふるさと納税の駆け込み | 食品ジャンルを前面に |
| 年末 | 大掃除・福袋 | 日用品ジャンルを前面に |

## 戦略④:複数サイト展開(慣れてきたら)

このサイトは**コピーして量産できます**。GitHubのリポジトリをもう1つ作り、`generate.py` の設定(サイト名とジャンル)だけ変えれば、10分で2号店が開店します。

- 1号店:総合(このまま)
- 2号店:コスメ特化 + コスメ系Xアカウント
- 3号店:アウトドア特化 + 趣味系Xアカウント

特化サイト×特化SNSの組み合わせを増やすほど、入口が増えます。

逆に、このサイトは**やめても腐らない**(自動更新が続く)ので、細く長く積み上げるのに向いています。

## やってはいけないこと

- ❌ 広告表記(「プロモーションを含みます」)を消す → 法律違反になります
- ❌ 「絶対痩せる」「最安値保証」など自分で効果・価格を盛る → 楽天規約・景表法違反
- ❌ 自分でリンクを踏んで自分で買う(自己アフィリエイトのつもりの不正クリック) → アカウント停止リスク
- ❌ 他人のレビューや記事のコピペをサイトに足す → 著作権侵害
